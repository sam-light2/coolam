COOLAM DEVELOPMENT — image assets
==================================

Place the following 7 image files in THIS folder (images/), using these
EXACT filenames (case-sensitive). The site references them by these names.

  kitchen.webp     About-section feature image (modern kitchen interior)
  Simmie.webp      Headshot, Simmie Cooper (Team section)
  Steven.webp      Headshot, Steven Lampert (Team section)
  3928.webp        Exterior photo, 3928 Hawthorn Ave (Properties)
  3930.webp        Exterior photo, 3930 Hawthorn Ave (Properties)
  4030.jpg         Exterior photo, 4030 Hawthorn Ave (Properties)
  4032.jpg         Exterior photo, 4032 Hawthorn Ave (Properties)

Notes
-----
- Until a file is present, the site shows a tasteful built-in placeholder in
  that slot (it will not look broken), then swaps in your photo automatically
  once the file exists at the path above.
- Filenames are case-sensitive on GitHub Pages / most servers: "Simmie.webp"
  is not the same as "simmie.webp".
- Recommended: kitchen.webp around 1600x1280 (landscape); headshots roughly
  square or portrait; house exteriors landscape (4:3 works well).
